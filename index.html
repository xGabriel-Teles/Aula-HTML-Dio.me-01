O que é HTML?

O HTML é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.



HTML é uma linguagem de programação?

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.



Criadores do HTML e suas versões

O HTML foi criado por Timothy John Berners-Lee e Dan Connolly.

Versões

  1. HTML 1.0 - Publicada em 1993
  2. HTML 2.0 - Publicada em 1995
  3. HTML 3.0 - Publicada em 1997
  4. HTML 4.0 - Publicada em 1999
  5. HTML 4.01 - Publicada em 2000
  6. HTML 5.0 - Publicada em 2014


Usos comuns

  • Desenvolvimento web

    Os desenvolvedores usam códigos HTML para projetar como um navegador vai exibir os elementos das páginas, como textos, hiperlinks e arquivos de mídia.

  • Navegação na internet

    Os usuários podem navegar facilmente e inserir links entre páginas e sites relacionados, já que o HTML é amplamente usado para incorporar hiperlinks.

  • Documentação

    O HTML torna possível a organização e a formatação de documentos, de maneira similar ao Microsoft Word.
